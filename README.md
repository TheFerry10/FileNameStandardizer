# FileNameStandardizer

## Overview
**FileNameStandardizer** is a Python utility for renaming and standardizing media files.

## Features
- Rename media files based on predefined patterns.
- Standardize file names to a simple timestamp format
- Rename jpg, mp4 from Android and Whatsapp format into a standardized format
